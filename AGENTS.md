# Repository Instructions

This repository is a small OpenFOAM CFD portfolio project.

Rules:

1. Do not fabricate simulation outputs, residual histories, centerline profiles, or figures.
2. Clearly distinguish between case setup and executed simulation results.
3. Demonstrate a traditional CFD workflow: mesh, boundary conditions, solver configuration, run logs, residual parsing, sampling, post-processing, and documentation.
4. Keep generated files small. Do not commit large OpenFOAM time directories unless necessary.
5. If OpenFOAM is unavailable in the environment, provide exact local reproduction commands.
6. `docs/cv_bullets.md` must contain only conservative resume bullets supported by actual files and outputs.
